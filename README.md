# Raxode

Raxode is the official Praxis coding-agent product: a backend plus the legacy terminal UI built on `@praxis-ai/praxis`.

## Install

```bash
npm install -g @praxis-ai/raxode
```

Node.js `>=24.15.0` is required.

## Usage

```bash
raxode login
raxode status
raxode
```

Raxode stores its default product configuration in `~/.raxode/config.json` and `~/.raxode/auth.json`.
